# Репозиторий к видео [Maven, Gradle и сборка проекта](https://youtu.be/Vf7awezFULc)

[![Превью видеоролика](http://lid.tv/images/video/preview/video-12-preview.png)](https://youtu.be/Vf7awezFULc)
